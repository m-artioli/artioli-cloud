Bem-vindo(a) ao meu repositório oficial da **Artioli Cloud**, uma plataforma de hospedagem de servidores focado no meu desenvolvimento e para do cliente gerando simplicidade de gestão de seus produtos.

## Objetivo
Projeto desenvolvido como portfólio Full Stack de ponta a ponta, visando a criação de um SaaS completo para hospedagem de jogos (Minecraft, GTA V, etc).

## Stack Tecnológica
- **Back-end:** C# .NET 10 (Web API)
- **Front-end:** React.js
- **Banco de Dados:** Microsoft SQL Server
- **Infraestrutura:** Docker & Integração com Pterodactyl (Planejado)

## Funcionalidades Principais (Roadmap)
- [ ] **Vitrine (Site):** Área do cliente, Faturas, Status Page.
- [ ] **Painel de Controle:** Iniciar/Parar servidores, Console em Tempo Real.
- [ ] **File Manager:** Upload de mods via interface web.
- [ ] **Database Wizard:** Criação de bancos MySQL com 1 clique.
